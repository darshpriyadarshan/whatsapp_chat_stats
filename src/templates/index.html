How to get the stats?

Firstly you have to export the chat from whatsapp and save it

How to export? Click here for official documentation by Whatsapp

Make sure to export chat without media

Now that you have Whatsapp chat with ____.txt ready, click upload on top right corner

Privacy?

The chat file uploaded is deleted just after the stats are prepared. So the contents of the file will not be stored on the server

Note:

Currently this doesn't support group chat
